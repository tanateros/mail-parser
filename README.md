<h3>Email-parser</h3>
<br />
Данный "модуль" поможет собрать почтовые адреса практически с любого сайта. Он сначала соберет все внутренние ссылки, запустит парсеры в фоне (не дожидаясь ответа будет проводить парсинг), покажет округленно сколько минут ждать до "решения" со всех парсов и через это время редиректнет автоматически на страницу решения, также добавит в базу решение и отобразит на главной
<br />
<h3>Требования и предустановки</h3>
<br />
Проект требует наличие PHP >= 5.4, Redis, Laravel 5.
<br />
Простейший вариант: установить Vagrant и добавить Homestead от Laravel 5 (http://habrahabr.ru/post/251529/),
<br />
в корневом каталоге создать каталог cache,
<br />
app\Http\routes.php заменить тем что в данном проекте,
<br />
добавить файл cli.php в app\Http\,
<br />
через Composer добавить PRedis
